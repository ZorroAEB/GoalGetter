Goal Getting Tips

  • Prioritize-make a list of your goals from most to least important and set your goal accordingly.
  • Set Specific Goals-make goals specific, as in, instead of saying that you will write, say that you will write 50,000 words
  • Set Meausurable Goals-set goals that you can measure like words or hours.
  • Set Relevant Goals-set goals that matter to you, and maybe to the world
  • Set Time Bound Goals-set goals that you have a time limit on, like a New Year's Resolution.
  • Set Achieveable Goals-set goals that will push you, but not push too hard.
  • Set Goals That You Can Control-don't set a goal like: I will change the weather, because you can't control the weather.
  • Manage Your Risks-don't be too risky, as in, don't say that you will climb Mount Everest if you have never climbed a tree.
